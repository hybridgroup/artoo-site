Communication with Devices

Socket to serial connection

Also known as Serial to network connection

Sometimes, when creating applications, developers are barred from accessing hardware devices such as serial ports, but have the need to access serial ports in order to communicate with interface boards and sensors.

This type of communication solves the problem by connecting on one side to a serial port and on the other side acting as a TCP/IP server.

With the Sphero, the initial connecting process might require a few retries, but once connected performance is fine.

Any OS

  • Turn on your bluetooth and device (sphero).
  • Connect the sphero to the computer using Bluetooth.
  • Run this command to look for it: 
    artoo connect scan
  • You will see the following output: 
    Scanning ...
00:06:66:44:39:C6   Sphero-YBW
  • Use that output to bind your device by name running this command: 
    artoo connect bind 00:06:66:44:39:C6 Sphero-YBW
  • Finally, run this command to connect to your device: 
    artoo connect serial Sphero-YBW 4567
  • You should see this message displayed: 
    N listening on AF=2 0.0.0.0:4567
  • Leave this terminal open and running.

MacOSX

  • Download and install Multicom.
  • Open the Multicom application and set the following configuration to Port A and Port B:
  • NOTE Port B information is very important to make sure that the listen port is the same one that you are using in your Ruby program; you could change this to fit your needs.
  • Both ports should be green.

Serial connection

MacOSX/Linux

  • To get the serial port identifier for your device, open a terminal and type the following command:

    ls /dev/tty.*

  • This command should list your device, for example 
    /dev/tty.Sphero-ORB-RN-SPP.
  • You should change the code to connect to that port, for example: 
    connection :sphero, :adaptor => :sphero, :port => '/dev/tty.Sphero-ORB-RN-SPP'
  • The latency on the initial connection is fairly high on all Bluetooth devices. The best way to get around this is to use a socket to serial utility to establish the connection.
