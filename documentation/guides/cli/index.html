A Guide by Humans, for Humans.

What is it?

Artoo CLI is a powerful command line interface tool that provides several tools to make RobotOps and setting up your devices to work with Artoo a lot easier.

How to install

Installing Artoo CLI is very easy, it is included with the artoo gem and more commands are added to it when you install other Artoo gems like artoo-arduino and artoo-digispark. 

$ gem install artoo

How can I use it?

The Artoo command line interface tools are pretty easy to use and designed to be intuitive, they also provide contextual help regarding the commands you are using. After you have installed the `artoo` gem, you just need to call the `artoo` command and a list of available commands (this list may vary if you have installed any of those other artoo gems we mentioned) will be printed to the terminal.

Printing a list of available commands

$ artoo
Commands:
  artoo bluetooth <SUBCOMMAND> ...ARGS    # Scans, pairs and Connects to a bluetooth device
  artoo console ROBOT                     # Run a robot using the Robi console
  artoo generate <SUBCOMMAND> ...ARGS     # Generates a new robot or adaptor
  artoo help [COMMAND]                    # Describe available commands or one specific command
  artoo install <SUBCOMMAND> ...ARGS      # Installs utility programs, like socat
  artoo scan <type> ...ARGS               # Installs utility programs
  artoo start ROBOT                       # Run a robot
  artoo version                           # Displays the current version

Scanning serial port for connected devices

$ artoo scan serial
[    0.000000] console [tty0] enabled
[    0.526190] 00:08: ttyS0 at I/O 0x3f8 (irq = 4) is a 16550A
[27425.430589] cdc_acm 3-4:1.0: ttyACM0: USB ACM device

artoo bluetooth <subcommand> <args>

Description

Lets you bind and connect to bluetooth devices by specifying a subcommand and providing the appropiate arguments, the avilable options for the subcommand are `pair`, `unpair`, `bind` and `connect`.

Description: binds a bluetooth device to a serial port. Notes: You might need to provide root access.

Description: unbinds a bluetooth device from a serial port. Notes: You might need to provide root access. 

$ artoo bluetooth unbind 00:06:66:4A:23:C3 Sphero-RPB

artoo bluetooth connect <name>

Description: connects a bluetooth device, that has been binded to a serial port, to a TCP socket using socat. Notes: You need to run `artoo bluetooth bind` before trying to connect using this command. Use the same name you use running `artoo bluetooth bind`.

That's ok, but... are those all the commands available?

Good you ask, and the answer is NO! Other Artoo modules include more commands that will prove very usuful and make your life easier when working in RobotOps, these will be made available for you when you install other Artoo gems, no extra work needed.

Ok, then show me more Artoo CLI commands.

The following is a list of commands available to you when you install the artoo-arduino and artoo-digispark gems, after installing, run the `artoo` command again (the list also includes the default Artoo CLI commands). 

$ artoo
Commands:
  artoo bluetooth <SUBCOMMAND> ...ARGS    # Scans, pairs and Connects to a bluetooth device
  artoo console ROBOT                     # Run a robot using the Robi console
  artoo arduino SUBCOMMAND <serial_port>  # All arduino related commands you'll need to work with artoo
  artoo firmata SUBCOMMAND <serial_port>  # Installs avrdude and uploads firmata to the arduino
  artoo generate <SUBCOMMAND> ...ARGS     # Generates a new robot or adaptor
  artoo help [COMMAND]                    # Describe available commands or one specific command
  artoo install <SUBCOMMAND> ...ARGS      # Installs utility programs, like socat
  artoo littlewire SUBCOMMAND             # Uploads littlewire protocol to digispark
  artoo scan <type> ...ARGS               # Installs utility programs
  artoo start ROBOT                       # Run a robot
  artoo version                           # Displays the current version

artoo-arduino CLI commands

artoo arduino <subcommand> [address]

Provides all necessary commands to use the arduino with artoo. 

$ artoo arduino
arduino commands:

  artoo arduino install                # Install avrdude in order to be able to upload fi...
  artoo arduino scan                   # Scan for connected arduino devices
  artoo arduino upload [ADDRESS]       # Uploads firmata to the arduino using avrdude

artoo firmata <subcommand> [address]

Provides several Firmata protocol utilities, these commands lets you install avrdude (needed to upload firmata to arduino) and upload the firmata protocol to arduino, that way you can connect and communicate with the arduino using Artoo.

  artoo firmata install           # Install avrdude in order to be able to upload firmata to the arduino
  artoo firmata upload [address]  # Uploads firmata to the arduino using avrdude

artoo firmata install

artoo-digispark CLI commands

artoo littlewire <subcommand>

In the same way that firmata lets you communicate with the arduino using Artoo, littlewire lets you communicate with the digispark, it is a communication protocol/firmware that can easily be uploaded to the digispark using Artoo. Let's check the CLI commands. 

$ artoo littlewire
littlewire commands:
  artoo littlewire help [COMMAND]  # Describe subcommands or one specific subcommand
  artoo littlewire set-udev-rules  # Sets necessary udev-rules in linux systems
  artoo littlewire upload          # Uploads littlewire to Digispark devices

artoo littlewire upload

Description: Uploads littlewire to the digispark (also installs the necessary udev-rules, only in linux systems). Notes: Do not connect the digispark until prompted to do so.
